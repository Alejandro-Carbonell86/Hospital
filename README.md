# Software Hospital - Login

Este proyecto contiene un formulario de login para un software de hospital.

## Estructura del proyecto

- `index.html` - Página principal del login.
- `css/style.css` - Estilos del formulario y la interfaz.
- `js/app.js` - Lógica de validación y mensajes en JavaScript.

## Cómo usar

1. Abre `index.html` en el navegador.
2. Ingresa un usuario y contraseña.
3. El login válido de ejemplo es:
   - Usuario: `admin`
   - Contraseña: `hospital123`

> Este proyecto usa validación en el navegador para practicar. En un sistema real, el servidor debe validar las credenciales.
